# DSA practice repo


| **Pattern**                    | **When to Use**                                             | **Keywords / Signals to Look For**                                                  | **Typical Data Structures** | **Example Problems**                                |
| ------------------------------ | ----------------------------------------------------------- | ----------------------------------------------------------------------------------- | --------------------------- | --------------------------------------------------- |
| **Sliding Window**             | Need optimal subarray/substring over a **contiguous range** | “subarray”, “substring”, “longest”, “shortest”, “at most k”, “window”, “continuous” | Array, String, HashMap      | Longest substring without repeats, Max sum subarray |
| **Two Pointers**               | Array/string, usually **sorted** or symmetric traversal     | “sorted”, “pair”, “triplet”, “target sum”, “palindrome”                             | Array, String               | Two Sum (sorted), Remove duplicates                 |
| **Fast & Slow Pointers**       | Detect cycles or middle of structure                        | “cycle”, “loop”, “circular”, “middle of list”                                       | Linked List                 | Cycle detection, Find middle                        |
| **Prefix Sum**                 | Repeated **range sum queries**                              | “range sum”, “sum between i and j”, “multiple queries”                              | Array, HashMap              | Subarray sum equals K                               |
| **Binary Search**              | Search in **sorted** or monotonic space                     | “sorted”, “minimum”, “maximum”, “first/last”, “log n”                               | Array                       | Search rotated array                                |
| **Binary Search on Answer**    | Optimize min/max under constraints                          | “minimize maximum”, “maximize minimum”, “capacity”, “days”                          | Array                       | Koko eating bananas                                 |
| **Hashing**                    | Need fast lookup / frequency counting                       | “frequency”, “count”, “distinct”, “lookup”                                          | HashMap, Set                | Anagrams, Two Sum                                   |
| **Stack**                      | Last-in-first-out logic                                     | “next greater”, “balanced”, “undo”, “monotonic”                                     | Stack                       | Valid parentheses                                   |
| **Monotonic Stack**            | Nearest greater/smaller elements                            | “next greater/smaller”, “previous”                                                  | Stack                       | Daily temperatures                                  |
| **Queue / Deque**              | FIFO processing or sliding window max/min                   | “level order”, “process in order”, “window max”                                     | Queue, Deque                | Sliding window maximum                              |
| **Breadth-First Search (BFS)** | Shortest path / level traversal                             | “minimum steps”, “shortest path”, “levels”                                          | Queue, Graph                | Shortest path in grid                               |
| **Depth-First Search (DFS)**   | Explore all paths / combinations                            | “all paths”, “connected components”, “explore”                                      | Recursion, Stack            | Islands count                                       |
| **Backtracking**               | Enumerate **all possibilities**                             | “all combinations”, “permutations”, “choices”                                       | Recursion                   | N-Queens                                            |
| **Dynamic Programming (DP)**   | Optimal solution with overlapping subproblems               | “maximum”, “minimum”, “number of ways”, “choices”                                   | Array, Map                  | Knapsack, Fibonacci                                 |
| **Greedy**                     | Local optimal leads to global optimal                       | “earliest”, “minimum intervals”, “maximum profit”                                   | Array                       | Activity selection                                  |
| **Union Find (DSU)**           | Connectivity & grouping                                     | “connected”, “groups”, “components”                                                 | Parent array                | Network connections                                 |
| **Heap / Priority Queue**      | Repeated min/max retrieval                                  | “top k”, “kth largest”, “stream”                                                    | Heap                        | Kth largest element                                 |
| **Bit Manipulation**           | Low-level optimization or uniqueness                        | “single number”, “xor”, “bit”                                                       | Integers                    | Find unique element                                 |
| **Graph Topological Sort**     | Dependency resolution                                       | “prerequisites”, “order”, “DAG”                                                     | Graph, Queue                | Course schedule                                     |
